How It Works

Every network packet passes through 3 layers. We peel each one.

Step 01

Open a Raw Socket

We create a special raw socket — unlike normal sockets that only see data meant for your app, this one captures every packet on the network interface, regardless of where it's going.

socket.AF_PACKET + socket.SOCK_RAW

Step 02

Parse the Ethernet Frame

The outermost wrapper is the Ethernet frame (14 bytes). It tells us the MAC addresses of sender and receiver, and what kind of data is inside (IPv4, IPv6, ARP…).

struct.unpack("! 6s 6s H", data[:14])

Step 03

Parse the IPv4 Header

Inside the Ethernet frame sits the IP header (20 bytes). This gives us the Source IP, Destination IP, TTL (time-to-live), and the protocol number (6=TCP, 17=UDP, 1=ICMP).

struct.unpack("! 8x B B 2x 4s 4s", data[:20])

Step 04

Parse TCP / UDP / ICMP

Finally, we parse the transport layer to extract port numbers, TCP flags (SYN/ACK/FIN), and the actual payload data — the real message being sent.

struct.unpack("! H H L L H", tcp_data[:14])

Step 05

Display + Log

Everything is colour-coded and printed to the terminal in real time. When you stop the sniffer, stats are summarised and a .txt log file is saved automatically.

netscope_log_YYYYMMDD_HHMMSS.txt

Key Concepts

This project teaches real cybersecurity fundamentals — not just syntax.

🔌
Raw Sockets
Bypass the OS filter and capture every single packet on the wire — the same technique used by tools like Wireshark and tcpdump.

📦
Packet Structure
Every packet is just bytes. Learn how Ethernet → IP → TCP layers are stacked inside each other like envelopes inside envelopes.
struct.unpack()
🌐
Network Protocols
Understand TCP (reliable), UDP (fast), and ICMP (control) — and know when the internet uses each one, and why.
TCP / UDP / ICMP
🚪
Ports & Services
Port 443 is HTTPS. Port 22 is SSH. Port 53 is DNS. Learn how each service has its own "door" on a computer.
20+ port mappings
🔢
Binary Parsing
Unpack raw bytes into numbers using Python's struct module — a skill used in reverse engineering and forensics.
big-endian bytes
🏳
TCP Flags
SYN, ACK, FIN, RST, PSH, URG — these flags control every TCP connection handshake. See them in real traffic live.
bitmask operations

Key Code Explained

The most important piece — parsing the IPv4 header, annotated for beginners.

parse_ipv4()
# ─── parse_ipv4: extract IP header fields from raw bytes ──────────────── def parse_ipv4(data): """ IPv4 header layout (20 bytes minimum): Byte 0 : Version (4 bits) + Header Length (4 bits) Byte 8 : TTL — how many hops before the packet is discarded Byte 9 : Protocol Number (6=TCP, 17=UDP, 1=ICMP) Bytes 12-15 : Source IP address Bytes 16-19 : Destination IP address struct format string explained: '!' = network (big-endian) byte order '8x' = skip 8 bytes (we don't need version/flags/etc) 'B' = 1 unsigned byte (TTL) 'B' = 1 unsigned byte (protocol) '2x' = skip 2 bytes (checksum) '4s' = 4 raw bytes (source IP) '4s' = 4 raw bytes (destination IP) """ version_ihl = data[0] ihl = (version_ihl & 0xF) * 4 # header length in bytes ttl, proto, src, dst = struct.unpack( "! 8x B B 2x 4s 4s", data[:20] ) src_ip = format_ip(src) # b'\xc0\xa8\x01\x01' → '192.168.1.1' dst_ip = format_ip(dst) return ihl, ttl, proto, src_ip, dst_ip, data[ihl:] # ─── format_ip: 4 bytes → dotted decimal ──────────────────────────────── def format_ip(raw_bytes): return ".".join(map(str, raw_bytes))

What Each Protocol Means

NetScope identifies these automatically and colour-codes them.

Protocol Number What it's used for Common Ports
TCP 6 Reliable delivery — web browsing, email, SSH, file transfers 80, 443, 22, 25
UDP 17 Fast, no handshake — DNS lookups, video calls, online games 53, 67, 161, 5353
ICMP 1 Control messages — ping, traceroute, "destination unreachable" No ports
GRE 47 Tunneling — wraps one protocol inside another (VPNs) No ports
ESP 50 Encrypted security — IPSec VPN traffic encryption No ports

Run It in 3 Steps

No pip install. No setup.py. Just Python 3 and Linux.

01
Clone the repository
git clone https://github.com/YOUR_USERNAME/CodeAlpha_NetworkSniffer
Or download the ZIP from GitHub and extract it.
02
Navigate into the folder
cd CodeAlpha_NetworkSniffer
Make sure sniffer.py is inside this folder.
03
Run with sudo (admin rights needed for raw sockets)
sudo python3 sniffer.py
Press Ctrl+C to stop. A log file will be saved automatically.
