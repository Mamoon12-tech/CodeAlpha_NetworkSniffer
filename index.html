Run It in 3 Steps

No pip install. No setup.py. Just Python 3 and Linux.

01
Save the file to your Linux machine
sniffer.py
Download sniffer.py and place it anywhere on your Linux system or WSL terminal.
02
Give it execute permission
chmod +x sniffer.py
This tells Linux the file is allowed to run as a program.
03
Run it with sudo
sudo python3 sniffer.py
Press Ctrl+C to stop anytime. Session summary and log file save automatically.

How It Works

Every network packet passes through 3 layers. We peel each one.

Step 01

Open a Raw Socket

We create a special raw socket that captures every packet on the network interface, regardless of where it's going.

socket.AF_PACKET + socket.SOCK_RAW

Step 02

Parse the Ethernet Frame

The outermost wrapper is the Ethernet frame (14 bytes). It tells us MAC addresses and what type of data is inside.

struct.unpack("! 6s 6s H", data[:14])

Step 03

Parse the IPv4 Header

Inside sits the IP header (20 bytes) — Source IP, Destination IP, TTL, and protocol number (6=TCP, 17=UDP, 1=ICMP).

struct.unpack("! 8x B B 2x 4s 4s", data[:20])

Step 04

Parse TCP / UDP / ICMP

Finally we extract port numbers, TCP flags (SYN/ACK/FIN), and the actual payload — the real message being sent.

struct.unpack("! H H L L H", tcp_data[:14])

Step 05

Display + Log

Everything is colour-coded and printed live. When stopped, stats are summarised and a .txt log file is saved automatically.

netscope_log_YYYYMMDD_HHMMSS.txt

Key Concepts

This project teaches real cybersecurity fundamentals — not just syntax.

🔌
Raw Sockets
Bypass the OS filter and capture every single packet on the wire — same technique used by Wireshark and tcpdump.
socket.SOCK_RAW
📦
Packet Structure
Every packet is just bytes. Learn how Ethernet → IP → TCP layers are stacked inside each other like envelopes.
struct.unpack()
🌐
Network Protocols
Understand TCP (reliable), UDP (fast), and ICMP (control) — and know when the internet uses each one.
TCP / UDP / ICMP
🚪
Ports & Services
Port 443 = HTTPS. Port 22 = SSH. Port 53 = DNS. Learn how each service has its own door on a computer.
20+ port mappings
🔢
Binary Parsing
Unpack raw bytes into numbers using Python's struct module — a skill used in reverse engineering and forensics.
big-endian bytes
🏳
TCP Flags
SYN, ACK, FIN, RST, PSH, URG — these flags control every TCP connection. See them live in real traffic.
bitmask operations
